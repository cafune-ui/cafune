# Icon - 图标

## 引入
```javascript
import { Icon } from 'components';
```
## 使用

```javascript
// 使用内置图标
<Icon icon="back" />
// 使用图片地址
<Icon icon="http://someicon.com/someicon.png" />
```

## 配置项
| 参数 | 说明 | 类型 | 默认值 |备选值 | 是否必须 |
| --- | --- | --- | --- | --- | --- |
| icon |  | string | - | - | ✅  |
| prefix |  | string | `'caf-icon'` | - | ❌ |
| color |  | string | - | - | ❌ |
| size |  | string | - | - | ❌ |
| tag |  | string | `'i'` | - | ❌ |
