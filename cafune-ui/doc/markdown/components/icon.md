# Icon - 图标

## 引入
```jsx
import { Icon } from 'components';
```

## 配置项
| 参数 | 说明 | 类型 | 默认值 |备选值 | 是否必须 |
| --- | --- | --- | --- | --- | --- |
| icon | 图标 | string | - | - | ✅  |
| prefix | 自定义前缀 | string | `'caf-icon'` | - | ❌ |
| color | 图标颜色 | string | - | - | ❌ |
| size | 图标尺寸 | string | - | - | ❌ |
| tag | 图标标签名 | string | `'i'` | - | ❌ |
| className | 自定义类名 | string | - | - | ❌ |
